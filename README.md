Интерактивная AI-экосистема для автоматизации B2B-продаж инженерных систем.
Генерация смет, 3D-двойники котельных, голосовой AI-ассистент и тепловые карты внимания клиентов.

🚀 Киллер-фичи (Roadmap 2026)
Spatial Computing: Интерактивные 3D-модели оборудования с поддержкой дополненной реальности (AR). Клиент может "поставить" котел в свою комнату через камеру смартфона.

Voice-to-JSON Pipeline: Голосовой AI-ассистент на базе Web Speech API, который общается с клиентом прямо внутри лендинга и пересчитывает смету "на лету" без перезагрузки страницы.

Zero-UI Лидген: Бот принимает сырые аудио-сообщения от менеджера (с объекта) и сам структурирует их в строгое коммерческое предложение.

Heatmap & Events Analytics: Трекинг глубины просмотра скролла и кликов по тарифам с моментальным уведомлением менеджера в Telegram.

Асинхронное ядро: Связка FastAPI + Redis + Celery Worker для бесперебойной обработки тяжелых AI-задач без зависания интерфейса.

🛠 Архитектура
Система построена на микросервисной архитектуре и развернута на Railway:

bot.py — Telegram интерфейс (AI-CRM) для инженеров.

web_server.py — FastAPI сервер для приема аналитики и RAG-запросов от клиентов.

celery_worker.py — Фоновый воркер, обращающийся к LLM (Google GenAI) и рендерящий HTML.

Upstash Redis — Брокер сообщений для очереди задач.

GitHub Pages — Бессерверный хостинг для сгенерированных клиентских лендингов.
